# Blog Post API

Hello!

This repository contains everything needed to interact with a simple blog post API.

The API is accessible via a simple Node.js server utilizing a SQLite database file
for persistent data.

# API Implementation

1. GET /posts
  * Endpoint: '/posts'
  * Method: GET
  * Response format: JSON
2. POST /post
  * Endpoint: '/post'
  * Method: POST
  * Response format: JSON
  * Parameters:
    * title (required)
    * body (required)
